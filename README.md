# PerformativeApply 🕵️

A personalized job finder app - your gateway to finding the perfect job match!

## Features

✨ **Discover Jobs** - Browse through curated job listings
💼 **Save Jobs** - Bookmark your favorite opportunities
👤 **Profile** - Manage your job preferences and skills
📱 **Mobile-First** - Optimized for mobile devices
🎨 **Clean Design** - Modern, intuitive user interface

## Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/jace54321/perfomativeapply.git
cd perfomativeapply
```

2. Open `index.html` in your browser or use a local server:
```bash
python -m http.server 8000
# or
npx http-server
```

3. Navigate to `http://localhost:8000` in your browser

### Deploy on GitHub Pages

1. Go to your repository settings
2. Navigate to "Pages" section
3. Select `main` branch as source
4. Your site will be live at `https://jace54321.github.io/perfomativeapply/`
